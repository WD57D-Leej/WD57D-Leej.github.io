Assignment Title: Skill Cards UI Design using HTML Box Layout and CSS Selectors

Assignment Description:

For this assignment, you will be creating a skill cards user interface (UI) using your knowledge of HTML box layout and CSS selectors. The skill cards UI will showcase your ability to design and layout information in a clear and visually appealing manner.

    Instructions:
  1. Create a new HTML file and name it "skill-cards.html".
  2. Inside the HTML file, create a container div element with an id of "skill-cards-container". This container div will hold all of the skill cards.
  3. Create at least four div elements with class names of "skill-card". These four div elements will represent four different skills that you possess.
  4. Inside each skill card div, create an image element that represents the skill. The image element should have a class name of "skill-image".
  5. Underneath the image element, create a heading element that displays the name of the skill. The heading element should have a class name of "skill-name".
  6. Underneath the heading element, create a paragraph element that describes the skill. The paragraph element should have a class name of "skill-description".
  7. Use CSS selectors to style the skill cards UI. Each skill card should have a border, a background color , and a box shadow . The skill image should be centered within the skill card, and the skill name and description should be positioned below the image.
  8. Use CSS box layout to arrange the skill cards. The skill cards should be displayed in a row, with equal spacing between each card. The container div should have a width of 80% and be centered on the page.
  9. Save your HTML and CSS files and submit them as a single zip file for grading.

Grading Criteria:

    Your skill cards UI will be graded based on the following criteria:
  • Correct use of HTML box layout and CSS selectors
  • Visual appeal and organization of information
  • Consistency in design and layout of each skill card
  • Proper use of file naming conventions and submission of files in a zip format
